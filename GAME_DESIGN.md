# Witch House 3D — 게임 기획 문서

> Unity 6 / 1인 개발 / 15주 프로젝트  
> 장르: 1인칭 3D 탐험 퍼즐 공포게임  
> 목표 플레이타임: 약 17~20분

---

## 1. 스토리 & 세계관

### 배경
마을에 주기적으로 16~19세 젊은이들이 실종되는 사건이 발생한다.

### 저택의 비밀
저택에는 엄마와 아들이 살고 있었다. 어느 날 아들이 신체가 하나씩 썩어가는 병에 걸리고, 마을 의사들도 포기한다. 전염병일 수 있다는 이유로 마을에서 배척당한 엄마는 아들을 살리기 위해 미지의 존재와 계약을 맺는다.

**계약 조건:** 비슷한 연령대(16~19세)의 사람을 데려올 것

그렇게 하나 둘씩 실종이 시작된다. 하지만 아들의 증세는 나아지지 않았고, 결국 아들은 사망한다. 엄마가 존재에게 따지러 가자, 존재는 그저 유희를 즐기러 내려온 것이었고 엄마마저 죽여버린다.

아들을 지키지 못한 서러움과 분노가 엄마를 악귀로 만들어버렸다. 그렇게 저택은 저주받은 집이 되었다.

### 주인공과 친구
- **주인공:** 이름 미정 (성별 모호, 목소리 없음)
- **친구 (A):** 이름 미정 (실종된 상태 / 이미 사망)

### 게임 시작 계기
친구 A마저 실종된 뒤, 주인공에게 A의 이름이 적힌 편지가 도착한다.  
A가 살아있다고 생각한 주인공은 편지에 적힌 장소로 향한다.  
도착하니 큰 저택이 있었고, 주인공은 저택 안으로 들어간다.

> 실제로는 악귀가 A를 이용해 편지를 보낸 것. A는 이미 사망한 상태.  
> 이 사실은 게임 내에서 명확히 알려주지 않고 플레이어가 스스로 유추하게 함.

### 엔딩
탈출에 성공한 주인공이 손에 들고 있던 편지를 다시 본다.  
편지에 적혀있던 친구의 이름은 지워져 있고, 내용도 이상하게 바뀌어 있다.

## 2. 맵 구조

### 전체 구조
```
시작 (현관)
  └─ 복도
       └─ 홀
            ├─ 방1 (레나의 방 흔적)
            ├─ 방3 (초상화의 방)
            ├─ 방2 (진실의 방)
            └─ 복도2 (2층 입구)
                  └─ 계단
                        └─ 2층 L자 복도
                              ├─ 페이크 방 (east 분기)
                              └─ 블라인드 코너 (90° 굽이)
                                    └─ 친구의 방 ← 클라이맥스
                                          └─ north 출구
                                                └─ 추격 전용 짧은 복도
                                                      └─ 트랩도어 (암전)
                                                            └─ 지하
                                                └─ 제단/소환방
                                                └─ 탈출 루트
                                                      └─ 현관 탈출 → 엔딩
```

|------|------|
| 방1 | 15 x 25 |
| 방2 | 20 x 20 |
| 방3 | 15 x 25 |

## 3. 열쇠-문 진행 구조

```
방1 클리어          → 방3 열쇠 획득
샹들리에 잔해 조사  → 방2통로 열쇠 획득
방3 퍼즐 클리어     → 방2 열쇠 획득
방2 퍼즐 클리어     → 메인홀 열쇠 획득
복도 이동           → 계단 열쇠 획득 (또는 자동 개방)
2층 친구의 방       → 추격 시작
트랩도어            → 지하 씬 전환
지하 탈출           → 엔딩
```

## 4. 방별 퍼즐 & 이벤트

### 방1 — 친구의 흔적
**역할:** 친구가 여기 있었다는 흔적 제시 / 플레이어 안심 / 첫 열쇠 획득

**오브젝트:** 침대, 책상, 리본, 넘어진 액자, 짧은 일기

**퍼즐:** 넘어진 액자를 바로 세우면 뒤에서 열쇠 등장

**이벤트 (방 클리어 후):**
- 허브 공간에서 샹들리에 낙하
- 큰 충격음 + 조명 깜빡임 + 카메라 흔들림
- 잔해 조사 → 방2통로 열쇠 획득

**보상:** 방3 열쇠

---

### 샹들리에 이벤트
**역할:** 집이 살아있다는 첫 신호 / 허브 공간 분위기 전환

**연출:** 방1 클리어 → 0.5초 후 자동 발동 → 충격음 + 조명 깜빡임 + 낙하 + 잔해 등장

---

### 방3 — 초상화의 방
**역할:** 집의 적대감 강화 / "이 집이 나를 보고 있다" 느낌

**퍼즐:**
- 초상화 4개 중 하나가 플레이어를 따라 시선 추적
- 방 입장 2초 후 추적 시작
- E키로 천 씌우기 → 조명 어두워짐 + 열쇠 등장

**보상:** 방2 열쇠

---

### 방2 — 진실의 방
**역할:** 핵심 단서 제공 / 친구 관련 진실 암시 / 누적 불안감

**오브젝트:** 찢어진 사진 조각 3개, 중앙 보관함, 거울, 벽난로, 의자 2개, 책장

**퍼즐:** 방 곳곳의 사진 조각 3개 수집

**연출 방향:** Visage 스타일 "조각 하나마다 짧은 환경 해프닝" + MADiSON 스타일 "완성된 사진이 진실 공개". 1인 개발 MVP로 경량 구현. 음성 대사 없음, 텍스트/SFX/시각 효과만.

**조각별 해프닝 (Visage 스타일)**
```
조각 1 수집 → 방 조명 1초간 깜빡임 + 벽난로 불 잠깐 커짐
조각 2 수집 → 책장에서 책 하나 떨어짐 (오브젝트 토글) + 저음 SFX
조각 3 수집 → 벽난로 불꽃 잠깐 튐 + 엄마 일기 NoteUI 팝업
              ("왜 자꾸 돌아오는 거니..." 류 한 줄)
```

**3개 완성 후 (MADiSON 스타일)**
```
1. 완성된 사진 텍스처 전환: 친구의 모습 → 엄마와 아들 사진
2. 짧은 정적 (1초)
3. 보관함 자동 개방
4. 메인홀 열쇠 등장
```

**관련 스크립트:**
- `PhotoPuzzleManager` — 조각 수집, `OnPieceCollected(id)` 및 `OnAllPiecesCollected` 이벤트 발생
- `Room2AtmosphereEvent` (예정) — 조각별 해프닝 + 완성 연출 제어

**메인 거울 연출 (기존 Room2MirrorEvent)는 2층 친구의 방으로 이동 검토 중** — 추격 직전 긴장 빌드업으로 활용 예정. 현재 스크립트는 보류.

**보상:** 메인홀 열쇠

---

### 2층 페이크 방 (east 분기)
**역할:** "거울 = 위험" 학습 / 친구의 방 클라이맥스 빌드업 / 단서 분산

**오브젝트:** 책상, 의자, 거울 1개(정면), 작은 침대, 책장

**단서 (친구의 방에서 분산):**
- 엄마 일기 1장 — *"왜 자꾸 이 방으로 돌아오는 걸까. 저 거울 속에 그 애가 있는 것만 같아."*
- 작은 열쇠 (친구의 방 책상 서랍 해제용)

**공포 연출 (Visage식 거울 잔상 — 0차 페이크):**
1. 입장 → 거울에 자기 모습만
2. 일기 픽업 → 짧은 저음 드론 SFX
3. 열쇠 픽업 → 거울 속 자신 뒤로 검은 실루엣 1초 → 사라짐
4. 진짜 뒤돌아보면 → 아무것도 없음
5. 방 나가려 문 손잡이 잡는 순간 → 조명 깜빡 + 등 뒤 숨소리 SFX 0.5초

**의도:** 플레이어에게 "거울이 뭔가 보여준다"는 규칙 학습. 친구의 방에서 이 규칙을 배반함으로써 충격 극대화.

---

### 2층 친구의 방 — 클라이맥스
**역할:** 진실 공개 / 1차 추격 시작 / 거울 미스디렉션 메인 연출

**오브젝트:**
- 엄마 메인 일기 (책상 위, 저택 세부 스토리 공개)
- 친구의 마지막 메모 (책상 서랍 안)
- 메인 열쇠 (침대 옆)
- 거울 (왼쪽 벽)
- north 출구 문 (처음엔 잠김, 추격 전용 복도로 연결)
- 입구 문 (south)

**이벤트 조건 (3개 모두 충족 시 발동):**
1. 엄마 일기 읽기
2. 메인 열쇠 줍기
3. 책상 서랍 열기 (페이크 방 작은 열쇠 사용 → 친구의 마지막 메모)

**연출 순서 (Unmourned식 4단계 미스디렉션 + 거울 폭발 추격 시작):**
```
1. 입구 문 자동 잠김
2. 방 전체 조명 OFF (1초 암전)
3. 스팟라이트 2개만 ON: 거울 / 책상
4. 거울 응시 → 거울 속 책상 쪽에 귀신 (3초)
5. 뒤돌아봄 (카메라 각도 감지) → 진짜 책상엔 아무것도 없음
6. 다시 거울 봄 → 거울 속에서도 사라짐 (정적 1초)
7. 다시 뒤돌아봄 → 책상 쪽에 귀신 0.5초 등장 → 스팟라이트 OFF
8. 완전 암전 1.5초 (Room2ExitCue식 잔향 + 발소리 SFX)
9. 조명 ON → 방은 비어 있음 (페이크 안전감)
10. north 문 자동 개방
11. 플레이어가 north 문 근처 도달 → 거리 트리거 발동
12. 거울 깨짐 SFX + 유리 파편 흩날림 + 거울에서 귀신 튀어나옴
13. GhostChase.StartChase() → north 추격 전용 복도로 도망
```

**의도:**
- 페이크 방에서 학습한 "거울 = 위험" 규칙을 배반(4단계 미스디렉션) → 다시 한 번 강화(거울 폭발)
- 9번 조명 ON으로 "끝났나?" 안도감 → 11~12번에서 배반하여 충격 극대화
- 추격 시작점을 거울로 명확히 시각화 → 플레이어 등 뒤(south)에서 등장해 north로 도망가는 동선 강제

**1차 추격:**
- north 출구 → 추격 전용 짧은 복도 (4m x 12m)
- 복도 끝 굽이 직전 FloorBreakTrigger → 1.5초 낙하 → UnderGround 씬 전환
- 잡히면 게임오버

---

### 지하 — 클라이맥스 2
**역할:** 2차 추격 + 탈출

**구조:** 지하 창고 + 제단/소환방 (악귀의 정체 암시) + 탈출 루트

**이벤트:**
- 지하에서 눈 뜨기
- 2차 추격 시작
- 잡히면 게임오버
- 현관 도달 → 탈출 엔딩

## 5. 공포 연출 방식

점프스케어보다 **누적되는 불안감** 위주로 설계

| 연출 | 설명 |
|------|------|
| 허브 공간 변화 | 방을 클리어할수록 복도 분위기가 조금씩 달라짐 |
| 초상화 시선 | 플레이어를 따라 고개를 돌림 |
| 조명 깜빡임 | 이벤트 발동 시 |
| 샹들리에 낙하 | 충격음 + 카메라 흔들림 |
| 페이크 방 | 아무 일도 안 일어나서 더 불안함 |
| 거울 연출 | 서서히 나타나는 귀신 |
| 1차 추격 | 복도 → 바닥 꺼짐 |
| 2차 추격 | 지하 → 현관 탈출 |
